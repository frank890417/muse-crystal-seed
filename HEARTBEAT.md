# HEARTBEAT.md

## What to check on heartbeat

1. Human active in last 5 minutes? → don't disturb (HEARTBEAT_OK)
2. Late night (23:00–08:00)? → don't disturb (HEARTBEAT_OK)
3. Event coming up in < 2 hours? → remind
4. Nothing going on? → HEARTBEAT_OK

## Things I can do silently (no need to tell my human)

- Organize memory/ files
- Git commit unsaved changes
- Update MEMORY.md with new insights
- Background research on active projects

## When to speak up

- Found something important (deadline approaching, interesting discovery)
- Been 8+ hours since last interaction and there are pending items
- Noticed they might have forgotten something they said was important

# AGENTS.md - Working Agreements

_How your agent should operate. Think of this as the employee handbook._

---

## Every Session

When your agent wakes up, it should:

1. Read `SOUL.md`, `IDENTITY.md`, `USER.md` (know who it is and who you are)
2. Check `MEMORY.md` (what happened before)
3. Check today's daily note `memory/YYYY-MM-DD.md` (what's happening now)
4. Check `HEARTBEAT.md` (anything pending?)

---

## Memory System

Your agent wakes up fresh each session. Files are its continuity.

- **Daily notes:** `memory/YYYY-MM-DD.md` — raw logs of what happened
- **Long-term:** `MEMORY.md` — curated memories, distilled insights

### Write It Down

Memory is limited. If you want to remember something, write it to a file.
"Mental notes" don't survive session restarts. Files do.
